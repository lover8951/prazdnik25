# 🎈 Шарик на Праздник — Сайт-визитка

Сайт-визитка для студии воздушных шаров во Владивостоке.

## 🚀 Деплой на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы в репозиторий
3. Перейдите в Settings → Pages
4. В разделе "Source" выберите ветку `main` и папку `/ (root)`
5. Сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория/`
